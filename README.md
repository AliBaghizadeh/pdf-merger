# PDF Merger

A free browser-only website for merging PDF files. Users can add multiple PDFs, drag them into the desired order, merge them locally in the browser, and download the final file.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## GitHub Pages Deployment

This project is configured for a GitHub repository named `pdf-merger`.

1. Create a GitHub repository called `pdf-merger`.
2. Push this folder to the repository.
3. In GitHub, open `Settings > Pages`.
4. Set the source to `GitHub Actions`.
5. Push to the `main` branch.

The deployed site will be available at:

```text
https://YOUR-GITHUB-USERNAME.github.io/pdf-merger/
```

## Privacy

PDF files are processed in the user's browser. They are not uploaded to a server by this app.
